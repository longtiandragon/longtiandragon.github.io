技术社区

CSDN

CSDN 作为中国最大的开发者技术社区，是许多初学者在技术学习道路上最早接触、也最容易上手的平台之一。尽管其界面广告较多，但在入门阶段，同学们仍能通过 CSDN 获取大量实用知识。
例如，后文提到的部分软件的安装与使用教程，也会通过**CSDN **上的博客文章获得清晰的指引。

CSDN官方网站

GitHub

GitHub是一个面向开发者的软件开发和版本控制平台，基于Git版本控制系统。它提供代码托管、协作开发和访问控制等核心功能。开发者可以在平台上创建仓库来管理代码，通过fork、pull request等机制参与开源项目协作。平台提供问题跟踪、代码审查、持续集成等工具，支持软件开发全流程。GitHub已成为全球开源项目的主要托管平台，也是开发者展示技能和参与协作的重要社区。
同学们可以从GitHub上下载很多开源的作品供学习,其安装包也都托管在GitHub平台上供用户下载使用。

GitHub官方网站

Gitee

Gitee（码云）是中国主要的代码托管平台，提供与GitHub类似的代码托管、协作开发等功能。对于初学者，可以从Gitee上学习以下内容：

  1. 通过阅读国内优质开源项目的代码了解实际开发流程；

  2. 尝试参与简单的开源项目，学习团队协作和代码管理；

  3. 利用平台上的教学资源掌握Git基本操作。

Gitee的界面和社区以中文为主，为初学者提供了更友好的入门环境。
Gitee官方网站

告别低效手抄笔记,用Markdown重塑学习笔记

Typora

Typora 是一款极简风格的 Markdown 编辑器，它的核心特点是采用「所见即所得」的编辑模式，让用户能够专注于内容创作而非语法标记。
Typora支持更换主题,优化视觉体验,是许多网课视频的资料展示都是用的Typora,可以提前熟悉一下

Typora采用Markdown语法,需要同学们自行学习常用的Markdown语法,具体的下载和使用教程,可以参考B站视频【大学生扫盲课】4 typora安装与配置 markdown语法说明

Obsidian

Obsidian 是一款以「链接」为核心的知识管理软件，它使用本地 Markdown 文件来创建个人知识库。

与其他笔记工具不同，Obsidian 的核心功能是建立笔记之间的双向链接，形成相互关联的知识网络。每当链接到其他笔记时，软件会自动生成反向链接，帮助用户发现知识间的隐藏联系。通过图谱视图功能，可以直观展示所有笔记的连接关系，像大脑神经元网络一样呈现知识结构。

软件支持丰富的插件生态，用户可根据需求添加日历、看板、表格等功能。所有数据以本地 Markdown 格式存储，确保用户对数据的完全控制权。这种基于链接的笔记方式，特别适合需要深度思考和构建知识体系的学习者、研究者和内容创作者。

Obsidian同样采用Markdown语法,需要同学们自行学习常用的Markdown语法,具体的下载和使用教程,可以参考B站视频Obsidian入门保姆级教程：20分钟轻松上手Obsidian！

Notion

**Notion **是一款创新的一体化工作空间应用，它重新定义了数字工具的使用方式。其核心价值在于将传统上需要多个独立软件才能实现的功能——包括笔记记录、知识库构建、任务管理、数据库等——完美整合在同一个平台中。

这款工具的革命性体现在其独特的"块"编辑器架构。所有内容（无论是文本、图片、列表、代码块还是嵌入式文件）都被视为独立的"块"，用户可以像搭积木一样自由拖拽组合，创造出完全符合个人需求的自定义系统。这种设计打破了传统文档的线性限制，实现了前所未有的内容组织灵活性。

**Notion **最强大的功能之一是其数据库能力。用户创建的每个表格都是一个功能完整的数据库，支持多视图展示：同一组数据可以瞬间在表格、看板、日历、列表或画廊视图间切换。例如，学生可以用表格视图管理所有课程信息，一键切换为看板视图跟踪作业进度，或转为日历视图掌握截止日期。

平台提供了丰富的模板生态系统，涵盖学习计划、课程表、项目跟踪等学习场景。这些模板与实时协作功能相结合，使学生能够高效地进行团队项目管理。所有数据都支持云端自动同步，确保在电脑、手机和平板等设备间实现无缝切换。

对于学生，Notion 不仅能作为笔记工具，更可以成为个人学习操作系统：管理选课信息、规划复习计划、协作完成小组项目、构建个人知识库。其高度可定制的特性让每个用户都能打造出最适合自己学习习惯的数字环境。

CSDN:Notion安装及汉化

总结

三者同样都是基于Markdown语法用来做笔记的工具,同学们可以根据需求选择适合自己的,下面是建议:

  1. Typora:学习成本较低,支持离线文档,操作简洁
  2. Obsidian:学习成本一般,支持离线文档,插件多,可优化用户体验,有知识图谱等功能可帮助联系笔记与笔记之间的关联
  3. Notion:学习成本较高,不支持离线,能管理笔记,任务,数据库等多功能于一体
